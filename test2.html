1111Navigation
Use j/k to move between blocks. Press Enter to edit the selected block in Vim. Exit Vim to refresh the block content. Press ? to show the shortcuts panel.
Heading 1
Heading 1 is for top-level sections. Use it to structure the document into major parts.
Heading 2
You can structure documents with three heading levels. Use ,bh1, ,bh2, and ,bh3 for hierarchy.
Heading 3
Heading 3 is useful for fine-grained sections. Use it sparingly for subtopics.
Three.js blocks
Three.js blocks are JS modules with THREE, scene, camera, renderer, and canvas pre-wired. Use them for real-time 3D scenes.
LaTeX blocks
LaTeX blocks render with KaTeX in a WebKit view. Edit the LaTeX source directly.
Map blocks
Map blocks run Leaflet JS with L, map, and tileLayer globals. Use them to plot points, shapes, and paths on a dark basemap.
Python render blocks
Python blocks render to SVG via __gvim__.renderer. They are rendered at runtime for export.
Notes
- No inline mixing; each block is its own unit. - Vim runs externally; GTK stays focused on layout.
